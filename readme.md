# ReadTime Chating

>一个实时聊天DEMO

## 包含：
- Android端
- WEB前端(静态、实时）
- 后端

## 原理：
Android端与后端通过Socket长连接，DEMO中使用8088端口
WEB前端和后端通过Socket.IO库(Ajax Polling、WebSocket)
